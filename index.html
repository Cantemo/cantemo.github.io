Portal Apps

We have a number of example Apps, plugins and code associated with Cantemo Portal.

Google Analytics

A small example plugin which adds Google Analytics functionality to Portal.

Google Maps

This example app extracts GPS coordinates from image metadata and displays a Google Map on the item page.

Extra Item Page Information

See how to create a very small App that will add some extra information on the Item Page for video and sequence items.

Portal Plugin Template

Dashboard Widgets

Portal 2.1 and later introduces the Dashboard which you can develop your own widgets for. Here are some example widgets.

RSS Feed Widget

Present an RSS Feed inside the Dashboard

Archive Framework Apps

Portal 2.1 introduces the Archive Framework which allows you to write your own apps to integrate with 3rd-party archive solutions. Below are some example apps provided by Cantemo.

File System Archive

An example app which allows you to archive files to a directory. No special archive solution is required so this can be used to archive to an external NAS or similar for example.
